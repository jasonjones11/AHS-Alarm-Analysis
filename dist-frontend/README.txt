AHS Alarm Analysis - Frontend Distribution 
============================================== 
 
This folder contains the complete frontend application ready for deployment. 
 
DEPLOYMENT INSTRUCTIONS: 
1. Copy this entire folder to your target server 
2. Ensure Node.js is installed on the target server 
3. Run 'start-frontend.bat' to start the application 
   OR run 'node server.js' directly 
 
The application will start on port 3000 by default. 
You can set the PORT environment variable to change this. 
 
FOLDER CONTENTS: 
- server.js: Main application server 
- package.json: Dependencies information 
- public/: Static assets (images, icons, etc.) 
- .next/: Built application code 
- start-frontend.bat: Windows startup script
